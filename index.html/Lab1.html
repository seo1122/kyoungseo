HTML Introduction

HTML is the standard markup language for creating Web pages.

What is HTML?

  • HTML consists of a series of elements
  • HTML elements tell the browser how to display the contents
  • HTML elements label pieces of content such as "this is heading", "this is a paragraph","this is a link", etc.

What is an HTML Element?

<tagname> Content goes here... </tagname>
Start tag Element content End tag
<p> My First Heading </p>
<br> none

Note: Some HTML elements have no content (like the <br> element).
